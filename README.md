# Doraemon GadgetVerse

A complete fan-made educational website project.

## Pages

- Home
- Adventure Game: Nobita's Rescue Mission
- Doraemon Pocket
- Gadget Lab
- Episode Library
- Missions
- Leaderboard

## Main Features

- Canvas game
- Dynamic pocket inventory
- Full CRUD for gadgets
- Full CRUD for episodes
- Mission system
- Score leaderboard
- LocalStorage persistence
- Deployable static website

## How to Run

Open `index.html` in your browser.

## Deployment

Upload the folder to GitHub Pages, Netlify, or Vercel.

## Note

This project does not host copyrighted Doraemon episodes. The episode section is only an informational guide.
